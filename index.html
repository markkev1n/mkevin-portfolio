About Me

Hi, I'm Mark Kevin, a BSIT graduate from Saint Louis University with almost 3 years of experience in web development, spanning my college years, freelance projects, and previous employment.

During my tenure as a web developer, I've specialized in WordPress, having built 15 websites, including the main company site and client projects.

My expertise extends to utilizing page builders like Elementor, WP Bakery, and Guttenberg, as well as implementing custom code for API integration. I've also honed my skills in website maintenance, including cPanel hosting management, backup procedures, and malware protection.

Furthermore, I focus on optimizing website performance, achieving high scores on tools like PageSpeed Insight, and ensuring responsiveness across various devices.

I'm dedicated to staying updated with the latest technologies and methodologies in web development to deliver top-notch solutions.
